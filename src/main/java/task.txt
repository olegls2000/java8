Создайте приложение в виде API по управлению автосалоном подержанных автомобилей
У него на балансе может содержаться до 8 ми автомобилей.
Автомобили могут быть грузовые и легковые. 
У грузовых есть : Дата выпуска, техническое состояние, производитель. Грузоподьёмность.
У легковых есть : Дата выпуска, техническое состояние, производитель и количество пассажиромест.
У каждого автомобиля можно высчитать стоимость при покупке автосалоном и стоимость при продаже в салоне, она зависит от многих факторов.
Оценочная стоимость при покупке автомобиля салоном:
Грузового: грузоподьемность*КоэфГруза - возрастАвтомобиля*КоэфВозраста+техничСостояние*КоэфТехнСостояния
Легкового: ПассажироМеста*КоэфПассажиромест - возрастАвтомобиля*КоэфВозраста+техничСостояние*КоэфТехнСостояния
ЗАДАНИЕ
Создайте интерфейс под названием автосалон со следующими методами:
  int bayCar( auto ); // метод принимающий машину в автосалон, в случае успеха  возвращающий сумму за которую удалось продать машину и списывающий с баланса автосалона средства, в случае неуспеха возвращающий -1;
  int sellCar( int i); // метод продающий машину из автосалона, цена продажи на 20 % боьше оценочной стоимости. В случае наличия машины на указанном месте метод возвращает сумму продажи и записывает деньги на баланс, в случае отсутствия возвращает 0 и выводит соответствующее уведомление.
String report () // метод выводящий на єкран отчет о состоянии автосалона список автомобилей и на каких они меслах, баланс.
Реализуйте указанный интерфейс.
Создайте иерархию классов автомобилей. 
Реализуйте в классах автомобилей соответствующие конструкторы для автоматического заполнения полей,
кроме поля дата, его нужно передавать через параметр. Для поля дата выпуска использовать LocalDate тип данных.
КоэфГруза =15, КоэфициентВозраста = 8, КоэфТехнСостояния = 100, КоэфПассажиромест = 150.
Производители “Honda”, “Mercedes”, “BMW”, “Toyota”. Техническое состояние от 1 до 10 , грузоподьемность от 200 до 1500.
Пассажироместа от 2 до 8.
Все зараннее известные значения ни в коем случе не  должны прописываться в логике программы, реализуйте их через константы.
